forces-assemble

forces-assemble backend

This backend supports custom clients that wish to implement the following basic behavior.

  • Users subscribe to various "channels".
  • A user wishing to notify other users pushes an event on a channel.
  • All subscribers to that channel receive a push notification with that event.
